How to use?
  • /blockcount | returns the blockcount
  • /apiinfo | Return basic information on the API like version
  • /hashrate | Return the current hashrate
  • /diff | Return the difficulty
  • /meminfo | Return information about the mempool
  • /blockchaininfo | Get info about the blockchain
  • /height/e.g 1 | Shows info about that block
  • /getblockhash/e.g 1 | Only retrieves the blocks hash
